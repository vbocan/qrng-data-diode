1. Secure Key Generator
   - Generate cryptographic keys (AES, RSA seeds, ECDSA) using quantum entropy
   - Output formats: hex, base64, PEM
   - Simple CLI demonstrating /api/random endpoint

2. Random Number CLI Tools
   - randint: Generate random integers in range [min, max]
   - randfloat: Generate random floats in range [0.0, 1.0]
   - randbytes: Output raw random bytes (binary or hex)
   - UUID generator: Quantum-backed UUIDs for distributed systems
   - Demonstrates basic gateway API usage patterns

3. Monte Carlo Pi Estimation
   - Estimate π using random point sampling in unit circle
   - Compare quantum vs pseudo-random convergence rates
   - Output: estimated π, error bounds, number of samples
   - Showcases statistical quality of quantum randomness

4. Traveling Salesman with Simulated Annealing
   - Solve TSP using simulated annealing with quantum random transitions
   - Accept/reject moves based on quantum random probability
   - Visualize solution evolution and convergence
   - Demonstrates quantum randomness in optimization algorithms

5. Knapsack Problem with Genetic Algorithm
   - Solve 0/1 knapsack using GA with quantum random mutations/crossover
   - Quantum selection for parent chromosomes
   - Compare convergence vs pseudo-random GA
   - Shows quantum randomness improving metaheuristic search

6. Password and Passphrase Generator
   - Generate secure passwords with custom character sets
   - Diceware-style passphrases using quantum word selection
   - BIP-39 mnemonic phrases for cryptocurrency wallets
   - Entropy estimation and strength indicators

7. Lottery and Fair Draw Simulator
   - Simulate lottery draws using quantum selection
   - Bias testing and statistical uniformity checks
   - Audit logs for transparency
   - Demonstrates provably fair randomness

8. Random Walk Simulator
   - 2D/3D random walks for diffusion modeling
   - Applications: particle physics, stock price simulation, Brownian motion
   - Export trajectories to CSV for analysis
   - Statistics: mean squared displacement, diffusion coefficient

9. Procedural Terrain Generator
   - Generate height maps using quantum-seeded Perlin noise
   - Output as grayscale PNG images
   - Applications: game development, geological simulation
   - Demonstrates unpredictability in procedural generation

10. Statistical Randomness Test Suite
    - Run NIST SP 800-22 statistical tests on quantum data
    - Compare quantum vs pseudo-random sources
    - Tests: frequency, runs, longest run, spectral, template matching
    - Visualize test results and p-values
    - Validates quantum randomness quality

11. Quantum Dice Roller
    - Simple dice simulator (d6, d20, custom sides)
    - Multiple dice with modifiers (e.g., 3d6+2)
    - Statistics tracking across rolls
    - Educational tool for tabletop gaming

12. Shuffling Algorithm Demonstrator
    - Fisher-Yates shuffle using quantum randomness
    - Shuffle playing cards, playlists, datasets
    - Bias analysis to prove uniform shuffling
    - Applications in cryptography and gaming
